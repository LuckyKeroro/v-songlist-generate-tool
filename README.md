# VUP 歌单管理工具

一个用于生成虚拟主播（Vup/Vtuber）歌单网页的管理工具。

## 功能介绍

- **歌曲元数据管理**：上传 Excel 文件批量导入歌曲信息，支持模糊匹配
- **多平台搜索**：支持 QQ 音乐和网易云音乐 API 自动获取歌曲详情
- **资源管理**：本地存储歌曲元数据，自动下载专辑封面
- **静态网站生成**：一键生成可部署的静态歌单网站
  - 玻璃拟态设计风格
  - 支持按歌手首字母分类、语言筛选、搜索
  - 点击歌曲一键复制点歌格式

## 简单使用教程

### 环境要求

- Node.js 18+
- npm

### 安装依赖

```bash
npm install
cd site && npm install && cd ..
```

### 启动管理工具

```bash
npm run admin
```

访问 http://localhost:3001 打开管理界面。

### 基本流程

1. **配置基本信息**：在管理界面填写主播名称、公告等
2. **上传图片**：设置背景图片和头像
3. **导入歌曲**：
   - 上传 Excel 文件（第一列歌名，第二列歌手）
   - 点击搜索获取歌曲详情
   - 保存到本地
4. **生成网站**：点击"生成静态网站"按钮
5. **部署**：将 `dist/` 目录部署到任意静态托管服务

### 可用命令

```bash
npm run admin      # 启动管理工具 (localhost:3001)
npm run dev        # 开发模式预览静态网站 (localhost:3000)
npm run build      # 构建静态网站
npm run generate   # 从 Resources 生成网站数据并构建
```

## 目录结构

```
├── admin/          # 管理工具 (Node.js/Express)
├── site/           # 静态网站源码 (React)
├── scripts/        # 构建脚本
├── Resources/      # 本地歌曲元数据存储
├── public/         # 配置文件和图片
└── dist/           # 生成的静态网站
```
